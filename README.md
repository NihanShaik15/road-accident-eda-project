# road-accident-eda-project
Road Accident Analytics and Severity Prediction (2022–2025)

This project focuses on analyzing Indian road accident data from 2022 to 2025 using Python and Exploratory Data Analysis (EDA).

The objective of this project is to identify accident patterns, accident-prone cities, weather impacts, traffic density effects, and accident severity trends. The dataset was cleaned, processed, and analyzed to generate meaningful insights that can help improve road safety and decision-making.

📊 Project Objectives
Understand the structure of road accident data
Perform data cleaning and preprocessing
Analyze accident severity distribution
Identify high-risk cities and accident hotspots
Study the impact of weather conditions on accidents
Examine traffic density and peak-hour accident trends
Generate visualizations for better understanding
Prepare the dataset for future SQL, Dashboarding, and Machine Learning tasks
